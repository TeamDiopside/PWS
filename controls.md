# Controls

|                Key | Control                           |
|-------------------:|:----------------------------------|
|               WASD | Drive car                         |
| Click / Arrow keys | Move camera                       |
|                  E | Follow selected car               |
|                [ ] | Cycle through cars                |
|              1 - 6 | Debug Menu                        |
|                  C | End generation                    |
|                  + | Add to max change                 |
|                  - | Subtract from max change          |
|                  P | Add to max generation time        |
|                  O | Subtract from max generation time |
|              SPACE | Reset car/camera position         |
